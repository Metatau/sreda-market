# SREDA.MARKET - Real Estate Platform

## Project Overview
Real estate marketplace with investment analytics, property search, and admin capabilities.

## Tech Stack
- Frontend: React + TypeScript + Maps
- Backend: Node.js + Express
- Database: PostgreSQL + PostGIS
- Infrastructure: Docker, Telegram Bot
- Maps: Vector tiles support
